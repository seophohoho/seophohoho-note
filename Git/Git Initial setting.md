## 사용자 정보
```console
$ git config --global user.name "NAME"
$ git config --global user.email youraddr@example.com
```
## 설정 확인
```console
$ git config --list
user.name=NAME
user.email=youraddr@example.com
color.status=auto
color.branch=auto
color.interactive=auto
color.diff=auto
```
